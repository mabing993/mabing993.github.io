2017年几大存储技术趋势

发表于 2017-01-12

【51CTO.com快译】尽管目前产值超1000亿美元的存储行业会有许多改进，但与2016年一样，我们在今年将只看到一项革命性技术：3D NAND，预计以色列的几家初创公司会有抢眼的表现。

固态硬盘(SSD)

SSD会继续迅速取代传统硬盘(HDD)，进入到PC和笔记本电脑甚至高端存储系统。虽然SSD比HDD来得小巧，但由于没有活动部件，让SSD要可靠得多、快速得多。现在甚至能提供比HDD还要大的存储容量，而且几乎所有SSD都拥有AES加密这一选项。

对所有存储硬件来说，今年最大的变化将是由2D NAND向3D NAND(层数越来越多)大规模转变，以获得更大的存储容量和更低的价格。产品来自所有闪存芯片生产商：英特尔/美光、三星、SK海力士、东芝、西部数据/闪迪，可能还有Powerchip。

U盘和便携式电话也因而成为受益者。此外，搭载NVMe的快速PCIe接口(带宽更高、延迟更低)会涌入市场，迅速取代6Gb SATA和6Gb/12GB SAS连接技术。

另一项大有希望的闪存技术是来自英特尔/美光的3D XPoint/Optane，价格不菲，但速度极快，在2015年8月宣布后应该会在2017年开始量产。

HDD

HDD的磁录密度极难增加。现在，唯一的方法就是往驱动器增添更多的磁盘(磁盘里面注满了氦)。我们继续观望混合HDD和叠瓦式磁记录(SMR)，二者最终在自2014年以来单位交付量不断下滑的全球市场会不会取得成功。

到时可能有机会听到厂商首次宣布推出基于热辅助磁记录技术(HAMR)的HDD。这种硬盘增添了激光器，用来提高磁盘上的磁录密度，但性能可能不见得提高。不管怎样，别指望在2018年之前见到实际产品面市。

这两种不同存储部件的每吉字节价格继续在很大程度上有利于机械硬盘;过去这几年，SSD和HDD的每吉字节价格都在下降，但SSD比HDD降得更快。由于NAND芯片供应紧张，以及HDD和部件供应紧张，这个趋势会暂时发生变化。

U盘

由于闪存芯片的存储容量提高，U盘的容量会达到2TB，成为可能取代外部HDD和SSD的一种选择。

MRAM

这是一项还没有得到普及的最新技术，由Everspin和Nantero等公司大力推广。

磁盘

这个市场继续在萎缩，由LTO(第7代LTO的原生容量达到6TB)“一家独大”，现在只用于大容量的冷存储，与大容量的近线存储HDD相竞争。WORM光盘出现了同样的趋势。

备份

结合用户的备份(HDD或存储子系统)和外面的云是最佳方法，将来也是。这意味着，将来数据总是会有三个副本，以防任何意外：主存储、本地和外部备份。

接口

正如之前所述，存储行业会纷纷采用PCIe/NVMe以及USB Type C。苹果首创的这一种技术将成为所有笔记本电脑、PC甚至智能手机的主接口，原因是其速度快。但是Thunderbolt远未死亡，其新的第三代支持40Gb/s带宽。SAS和SATA会与光纤通道一样日渐式微，即便引入了速度达到32Gb/s的新版本。以往光纤通道市场的三大领导厂商：博科、Emulex和QLogic已先后被收购。

大有希望的市场和应用：

融合及更高一级的超融合是两大趋势。由于可扩展性、成本和存储效率等方面的优点，我们会继续看到超融合存储的采用得到大幅提高，Nutanix在去年9月首发上市(IPO)后一路领跑。

全闪存阵列继续是成长型市场，与闪存争夺软件定义层这一地位。结合SSD和HDD的混合配置会是一个更庞大的部分。

整体式存储子系统被价格更低的横向扩展型架构所取代。这种架构包括使用快速千兆网络的几个节点(配备HDD或SSD的服务器，可根据需要添加磁盘)。此外软件管理整个系统，不包括昂贵的磁盘阵列控制器，因为数据的一个或多个副本分散在不同节点上。

数据分析和人工智能将利用已存储的数据创造价值。

Docker容器成为行业的一部分。

公共云存储是存储领域发展最快的行业之一，领导厂商包括亚马逊AWS、谷歌云存储和微软Azure。

存储行业最有希望的市场之一就是视频行业，包括视频监控和后期制作，图像清晰度提高需要庞大的存储容量。
