码农列传

发表于 2017-01-04

太史公曰：码农者，其行或放荡不羁，或空室蓬户，或褐衣疏食，然其言必信，其行必果，已诺必诚，不爱其躯，赴士之厄困，既已存亡死生矣，而不矜其能。码之大者，为国为民。今取其一二分而记之。

运维工程师

滚滚长江东逝水，浪花淘尽，英雄代码中。
系统昨夜又抽风，需求变更，愁意浓，睡意空。
Bug故障夜里凶，任务报警满江红。
闻鸡起舞奋力勇，事了回首，夕阳正红。
善：降龙十八掌。此掌法精要，威力无穷。一招双龙取水，气走阴跷阴维，双机房断电演练让小子胆颤心惊；一招飞龙在天，气走督脉，行手阳明，数千服务器即刻重启，所有异常灰飞烟灭。降龙二十八掌’的精义，乃是‘有余不尽’四字，一掌之出，必须留有余力。非不得时，不得出，一出必击。

Web工程师

君不见交互美化数杯茶，浏览兼容生华发。
君不见像素字体美如画，加班熬夜啃锅巴。
js web 泪双流，css调参浑浊了双眸。
善：百变千幻云雾十三式。此衡山派绝学，招式千变万化，眩人耳目，页面成时，如梦如幻，使人深入其境无法自拔。

Java 工程师

心中藏风骚编程思想, Apache google各类jar包府中武库，
胸中装无上设计模式, SOA MVC 各种pattern 脑中书房。
善：达摩十八手。混元一气为正宗，万法皆从此中出，诸巧尽从里边生。十八手涵盖封装、抽象、继承、多态、接口、异常、覆盖、修饰、静态、动态、构造、方法、变量、常量、类型、参数、编译、运行。此外又从这18手中生出无穷变化，习练精熟者，刚柔相济，虚实并兼，行似游龙，动如闪电。

算法工程师

运筹帷幄之中，决胜千里之外。
统帅何须言军马？胸中自有雄兵百万。
善：八卦掌。代码氤氲变化，奥妙莫测，算法精微奥妙，其变无穷，有神化不测之机，其数式数形，纵横联络变化，溯其源，皆出理数之原，至妙之道。

数据库工程师

配置调试一手搞定，基础原理牢记心头。
Sql脚本性能最优，大促处理有我不愁。
善：四象掌法。此峨嵋武学源出河洛之图，变化方位暗合四象玄机 ,实为增删改查四字真诀，此招一经施展，万千玄象皆收眼底，百转千回尽刻归一。

ETL工程师

ETL工程师者，大数据搬运工也。
善：乾坤大挪移。其主旨乃在颠倒一刚一柔、一阴一阳乾坤二气，随意而行。
执手中数行Sql即可将亿万数据来回腾挪，至若切中肯綮，则批郤导窾，游刃有余，官止神行，事毕善刀而藏，不留名姓。

C++工程师

怅寥廓，问苍茫大地，谁主沉浮。
指点江山，激扬文字，粪土当年万户侯。
善：碧针清掌。此这路掌法能以内力鼓荡松针，代码敲动之时，成千上万枚指针指引对象，内力不继者观之这一切如同幻影，功法轻灵迅猛。操纵者内力虽非有形有质，却也已隐隐有凝聚意。然操纵不当，指针易反噬己身。

或曰：阿里既成，万事既理，天下举首而望：“马总之功也”，后之人循迹而慕，然工程狮程序猿百执之辛劳，而不得稍纪，令人喟叹。

臆想公司初起，诸码农请缨入内，涉SARS之流而建淘宝，去IOE之后而壮支付宝, 搭云梯而成阿里云，重技术而成中台。天下网商故为之三分，且阿里独大。

浩浩乎惊风扑面，凛凛乎黄尘刺目，大促梦回，寒星在空，键盘交飞，交易速成，双十一天下惊叹，
程序猿筚路蓝缕，前赴后继，方有今日之功。今做此文以表，望人怜之敬之。
